*todo-nav.txt*  Navigate identifier-keyed file-based TODO systems

==============================================================================
INTRODUCTION                                                        *todo-nav*

todo-nav jumps from a TODO identifier under the cursor (or a visual selection)
to its workstream entry or its index entry, in the current pane or a floating
dialog. Requires ripgrep (`rg`) on PATH; see |todo-nav-health|.

==============================================================================
SETUP                                                         *todo-nav-setup*

>lua
    require("todo-nav").setup({})
<
Options (defaults shown): >lua
    {
      todo_dir = "docs/ai_todos",
      index_file = "INDEX.md",
      exclude_glob = { "*.archive.md" },
      id_pattern = "[A-Z][A-Z0-9]+%-%d%d?%.?%d*",
      task_boundary = "^%d+%.",
      highlight = "Visual",
      float = { width = 0.8, height = 0.8, border = "rounded" },
      root = nil,
      keys = {
        task_pane = "<leader>gt",
        task_float = "<leader>gT",
        index_pane = "<leader>gi",
        index_float = "<leader>gI",
      },
    }
<

==============================================================================
MAPPINGS                                                   *todo-nav-mappings*

                                                  *<Plug>(TodoNavTaskPane)*
                                                  *<Plug>(TodoNavTaskFloat)*
                                                  *<Plug>(TodoNavIndexPane)*
                                                  *<Plug>(TodoNavIndexFloat)*
Bind these when you disable a default key (set its `keys` entry to `false`): >lua
    vim.keymap.set({ "n", "x" }, "<leader>tt",
      "<Plug>(TodoNavTaskFloat)", { remap = true })
<

==============================================================================
COMMANDS                                                   *todo-nav-commands*

:TodoNavTask     Preview the task under the cursor in a float.
:TodoNavIndex    Preview the index entry for the task under the cursor.

==============================================================================
HEALTH                                                       *todo-nav-health*

Run `:checkhealth todo-nav` to confirm ripgrep is available and the module
loads.
